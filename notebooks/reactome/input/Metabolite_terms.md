# Metabolite serch

## Vitamin A - retinol
- Generic term: Vitamin A = CHEBI:12777
- Common pathway form: all-trans-retinol = CHEBI:17336

## Vitamin C  
Generic term: Vitamin C = CHEBI:176783
Neutral acid form: L-ascorbic acid = CHEBI:29073
Common pathway form: L-ascorbate = CHEBI:38290


## Vitamin E:
Generic ChEBI term: CHEBI:33234 = vitamin E. ChEBI defines this as the class of vitamin E vitamers, including tocopherols and tocotrienols.
Common pathway form: CHEBI:22470 = α-tocopherol (alpha-tocopherol). ChEBI classifies α-tocopherol as a vitamin E member.

## Glutathione
- CHEBI:16856. ChEBI defines it as the neutral tripeptide compound.
- Common pathway form: glutathionate(1−) = CHEBI:57925. ChEBI explicitly describes this as the major species at pH 7.3, which is why it is often the best pathway-normalized form.
- CHEBI:177535, L-Glutathione (reduced)

## Alpha-lipoic acid
- ChEBI ID: CHEBI:16494 = lipoic acid. ChEBI lists “alpha-lipoic acid” as a synonym of lipoic acid.
- Common pathway form: CHEBI:30313 = lipoate. ChEBI explicitly says this is the major species at pH 7.3, so it is the best match for the physiological/pathway form.
- free small-molecule “common pathway form,” lipoate = CHEBI:30313

## Dihomo-γ-linolenic acid (DGLA)
Dihomo-γ-linolenic acid (DGLA) is a 20-carbon omega-6 polyunsaturated fatty acid, also called all-cis-8,11,14-eicosatrienoic acid.
- Neutral acid: DGLA = dihomo-γ-linolenic acid = CHEBI:53486
- Common pathway form: dihomo-γ-linolenate = CHEBI:71589
- Dihomo-gamma-linolenic acid ethyl ester = CHEBI:183405 

## cis-Aconitic acid
Neutral acid: cis-aconitic acid → CHEBI:32805
Common pathway form: cis-aconitate(3−) → CHEBI:16383

## taurine
- CHEBI:15891 = taurine.
- Common pathway form: CHEBI:507393 = taurine zwitterion. ChEBI describes this as the major species existing at physiological pH, which makes it the best pathway-normalized form.
- CHEBI:32970 - conjugate acid of 2-aminoethanesulfonate

## FIGLU (Formiminoglutamate)
- ChEBI ID: CHEBI:7274 = N-formimidoyl-L-glutamic acid. This is the neutral acid form and corresponds to FIGLU / formiminoglutamate.
- Common pathway form: CHEBI:18327 = N-formimidoyl-L-glutamate(2−). This is the conjugate base commonly used in biochemical reaction/pathway representations.

## Lipid peroxides - a mixture/class
- CHEBI:61051 — lipid hydroperoxide
- Pathway component: Lipid-OOH / lipid hydroperoxide
- hydroperoxy fatty acid (CHEBI:64009)
